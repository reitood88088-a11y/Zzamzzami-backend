**사용자 시나리오**

**진입** : 하루 일과가 끝나고 집에서 외국어 자격증 공부를 한다. 공부를 하고 마친다. 내일 복습을 위해 앱을 켠다.

처음 앱을 열면 앱의 홈화면이 보인다. 홈화면엔 사용자가 스캔한 데이터를 언어별로(영어, 중국어, 일본어), 날짜별로(가장 최신 날짜가 가장 왼쪽에 보이게, 가로로 일렬 정렬) 정리되어 있다.

**탐색** : 홈앱에서 스캔 버튼을 눌러 카메라를 활성화한다.

**입력** : 공부한 텍스트에 카메라로 스캔한다. 촬영하면 바로 텍스트를 인식하여 저장한다. 노트를 들고 다니는 것보다 간편하다!

**결과** : 중간 중간 자투리 시간에 홈탭에 ‘복습’ 버튼을 눌러서 어제 내용을 복습을 한다. ‘복습’파트는 어제 스캔한 내용 그대로 보여주고 주요한 단어와 표현 가르친다. 이때 형식은 인스타그램 릴스 스타일. 아래서 위로 손가락으로 넘긴다.

그리고 ‘AI퀴즈’으로 넘어간다. ‘AI퀴즈’은 인스타그램 스토리 스타일로 정답을 고르면 옆으로 넘어간다. AI가 어제 공부한 어휘를 다른 형식으로 문제들을 만들어 제공한다.  인스타그램 릴스와 스토리 형식 UI/UX을 비슷하여 친숙하다.
**이탈** : 앱의 모든 학습과정을 마치면 ‘학습 분석’탭에서 학습 데이터를 확인하며 앱을 종료한다. 공부한 내가 뿌듯하다.
