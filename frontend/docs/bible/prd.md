**1.제품 정의서 (PTS, 엘리베이터 피치)**

**Problem** - 기존 외국어 자격증 학습 앱들은 자체 제공하는 단어와 문장만 공부하도록 설계되어 있어, 정작 사용자가 개인적으로 공부하는 교재와 연동되지 않음. 그래서 복습 공부는 노트를 들고 다니면서 복습해야 한다.

**Target** - 외국어 자격증 취득을 목표로 공부 중인 취업 준비생

**Solution** - 내가 공부하는 외국어 교재 텍스트를 사진으로 찍기만 하면, AI가 나만의 다회독 복습 콘텐츠를 자동으로 생성해 주는 주동적 맞춤형 학습 서비스



**엘리베이터 피치**

사용자가 스캔한 교재 텍스트 이미지를 기반으로 AI가 맞춤형 다회독 콘텐츠를 자동 생성하는 '사용자 주도형 외국어 복습 솔루션’

좀 더 구체적으로 말하자면, 사용자가 교재 텍스트를 스캔하면,  텍스트 속 주요 단어와 어휘를 추출하여 복습하고, AI를 활용하여 복습한 단어를 변형문제로 풀어볼 수 있다.
